Detailed Description

Input/Output structure for the orient() function. The mesh field will not be modified during orientment. Instead, the translation and rotation fields will mark the needed transformation for the polygon to be in the orientd position. These can also be set to an initial offset and rotation.

The bed_idx field will indicate the logical bed into which the polygon belongs: UNORIENTD means no place for the polygon (also the initial state before orient), 0..N means the index of the bed. Zero is the physical bed, larger than zero means a virtual bed.
