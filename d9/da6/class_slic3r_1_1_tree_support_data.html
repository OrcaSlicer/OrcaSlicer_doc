Creates the areas that have to be avoided by the tree's branches in order to reach the build plate.

The result is a 2D area that would cause nodes of radius radius to collide with the model or be unable to reach the build platform.

The input collision areas are inset by the maximum move distance and propagated upwards.

tbb::concurrent_unordered_map<RadiusLayerPair, ExPolygons, RadiusLayerPairHash, RadiusLayerPairEquality> Slic3r::TreeSupportData::m_collision_cache
mutable

Caches for the collision, avoidance and internal model polygons at given radius and layer indices.

These are mutable to allow modification from const function. This is generally considered OK as the functions are still logically const (ie there is no difference in behaviour for the user betweeen calculating the values each time vs caching the results).

coconut: previously stl::unordered_map is used which seems problematic with tbb::parallel_for. So we change to tbb::concurrent_unordered_map
